Резюме

Ключевые навыки

BI & Визуализация

  • Инструменты: Tableau, Power BI (Мультимеры - switch, закладки), Redash, Qlik Sense, Google Looker.

Анализ данных и программирование (Data Engineering)

  • Python:
    • ETL-автоматизация: Разработка пайплайнов для выгрузки данных из CRM через API, обработка (pandas, numpy, собственные алгоритмы), загрузка в БД/хранилища.
    • Airflow: Создание DAG's для обработки задач (cron расписания, обработка ошибок, мониторинг). Пример: «DAG для ежедневного обновления HR-метрик из CRM».
    • Парсинг: Scrapy, BeautifulSoup (веб-скрапинг), REST API (интеграции с сервисами). Пример: «Сбор резюме с работных сайтов».
  • SQL:
    • Сложные запросы (CTE, подзапросы), оконные функции, оптимизация производительности.
  • Excel:
    • Обработка в Power Query, Power Pivot (модели данных, DAX)
  • Статистика:
    • Основы проверки гипотез (A/B-тесты, t-критерий, хи-квадрат), работа с распределениями эмпирически (Bootstrap генерации).

Метрики:

  • HR: Текучесть, ROI рекрутинга, истощение, Occupancy, Utilization и другие метрики колл центра.
  • E-commerce: Конверсии, churn, rfm, retention (скользяща, n-дня), cohort-анализ.
  • Финансовый анализ: Расчет выплат/начислений, операционная эффективность (CAC, LTV, ROI), прогнозирование.
